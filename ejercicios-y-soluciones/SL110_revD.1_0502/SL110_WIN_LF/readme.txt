README

SL-110: Fundamentals of the Java Programming Language

- The SL110_SOL_LF/exercises folder contains solutions for all exercises 
  in this course. It is recommended that students do their work in the 
  folder titled SL110_SOL_LF/exercises/[module_title].
  
  Instructions are included in the Student Guide and the Instructor Guide
  at the appropriate points within the course.

- The SL110_SOL_LF/examples folder contains all *complete* code examples 
  from the student guide and instructor guide. 

  Where necessary, subfolders are used within the folder for each module,
  if two or more classes are different but have the same name.
